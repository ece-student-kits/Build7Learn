# ATmega328P

The ATmega328P is an 8-bit AVR microcontroller used in Arduino Uno.

## Key Features

- 32 KB Flash Memory
- 2 KB SRAM
- 1 KB EEPROM
- 16 MHz Clock
- ADC, PWM, SPI, I2C, USART
- Low Power Modes
